# Immo Admin React

Interface d'administration pour la gestion immobilière développée avec React + Vite.

## Prérequis

- Docker
- Docker Compose

## Installation

```bash
# Cloner le projet
git clone <url-repo>
cd cours1/immo-admin-react

# Démarrer le container
docker-compose up -d
```

Les dépendances npm sont installées automatiquement au démarrage.

## Accès

- Application : http://localhost:3000

## Arrêter le projet

```bash
docker-compose down
```
