# Immo Client Vue

Application cliente pour la consultation immobilière développée avec Vue 3 + TypeScript + Vite.

## Prérequis

- Docker
- Docker Compose

## Installation

```bash
# Cloner le projet
git clone <url-repo>
cd cours1/immo-client-vue

# Démarrer le container
docker-compose up -d
```

Les dépendances npm sont installées automatiquement au démarrage.

## Accès

- Application : http://localhost:3001

## Arrêter le projet

```bash
docker-compose down
```
